STICKY NOTES!!

This allows developers to generate a sticky note on their page

  • By clicking '-' button, the sticky note would disappear and left only '+' button. By clicking '+', the sticky note would come back
  • 'U' is for underline, 'I' is for italic, and 'B' is for bold
  • By draging the sticky note's title, user can move sticky note around.
Documentations

Examples

  1. The normal sticky note const sn2 = new stickyNote() sn2.create('two', 200, 300, '') sn2.changePosition(450, 50)
  2. The sticky notes with other properties and in bullet point form:
    This can be used for student to answer a question const sn1 = new stickyNote() sn1.create('one', 200, 300, 'bullets') sn1.changeColor("green") sn1.changeTitle("Question 1: Introduce yourself") sn1.changePosition(450, 450) sn1.changeSize(400, 300) sn1.changeFontSize(20)
  3. Another Sticky note for numbering form:
    This can be used to write down comments const sn3 = new stickyNote() sn3.create('three', 200, 300, 'numbers') sn3.changePosition(450, 950) sn3.changeColor("lightBlue") sn3.changeFontColor("black") sn3.changeTitle("Write down what you think")
