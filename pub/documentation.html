STICKY NOTES DOCUMENTATIONS

Getting Started

  • This library is depending on jQuery. So the following scripts are necessary for it.
    • <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script> <script defer type="text/javascript" src='js/stickyNote.js'></script>
  • Create. There are three arguments needed. The first one is the id of the sticky note, the second and third ones are the width and height. The last one may have three different string: '', 'bullets', and 'numbers' to decide the type of sticky note.
    • const sn = new stickyNote() sn.create('id', 200, 300, 'bullets')

APIs

  • Change color. It need a color string as the argument. It will change the background color of the sticky note
    • sn.changeColor("green")
  • Change title. It need a title string as the argument. The default value of the sticky title is 'Sticky Note'. By calling this methid, the title can be changed to other text.
    • sn.changeTitle("Write down what you think")
  • Change position. It need a x position and a y position. The sticky note's top left corner will move to the new position.
    • sn.changePosition(450, 450)
  • Change size. It need two arguments, height and width. The sticky note would change its height and width to the new values.
    • sn1.changeSize(400, 300)
  • Change font size. It need a integer as the new font size. The words in the sticky note, including title and users' input, would have the new font size value.
    • sn1.changeFontSize(20)
  • Change font color. It need a color string as the argument. It would change the fonts in the sticky note to have use the new color.
    • sn3.changeFontColor("black")
